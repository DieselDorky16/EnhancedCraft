This mod requires Forge 1.15.2-31.2.0 or later.
Please note this is the experimental version of the mod. You can find the stable version of the mod on PlanetMinecraft under the same name.

How to install:

Step 1: Download the mod
Step 2: Make sure you have Forge installed, and make sure you launch it at least once, then close it.
Step 3: After closing Minecraft with Forge, drag and drop the downloaded file to your mods folder.
Step 4: Run Minecraft with Forge.

You can report bugs on the Github Issues page.
